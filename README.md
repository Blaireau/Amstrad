# Amstrad
Let's try to learn something new :)

# Tools used

- ACEDL pour l'émulation
- rasm pour la compilation
- vsCode pour l'édition de code

# Compilation

--> rasm.exe *filename*.asm

# Exécution

--> acedl.exe *filename*.dsk -autoRunFile *filename*.bin

pour le filename du disk et du bin vérifier la dernière ligne du code asm.
